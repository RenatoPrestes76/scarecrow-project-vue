## Instalação do projeto

```
npm install
```

### Rodar o projeto no navegador

```
npm run serve
```

### Desafios:

1. Criar os componentes:
   1.1. Header
   1.2. Title
   1.3. Paragraph
   1.4. Button
   1.5. Footer

2. Trocar o texto interno por "props" e escrever o texto quando o componente fora chamado.
